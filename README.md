# FPGA_Image_Processing_Project

---

## 요약
- 본 프로젝트는 **OV7670 카메라 모듈**과 **VGA** 출력의 동작 원리를 심층적으로 이해한 뒤, 수집된 영상 처리 기법을 적용하여 실시간 교통관리 시스템을 구현하는 것을 목표.
- 최종적으로는 교통 흐름 감시, 이상 상황 탐지, 신호 제어 연계 등 실무에 적용 가능한 시스템 개발을 목표로 함.
